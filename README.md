# TF2BD-Database
My Database of Cheaters and Suspicious Players in Team Fortress 2.

---
I’m currently using [this](https://github.com/surepy/tf2_bot_detector), which is a fork of [Pazer TF2BD](https://github.com/PazerOP/tf2_bot_detector). If you know of a better, actively maintained fork of Pazer TF2BD, let me know.

**Q:** *How can I use your TF2 Cheaters Database?*

**A:**

**Option A:** Download the ZIP file of this repository by clicking the `Code` button, then selecting `Download ZIP`. After extracting it, all you need is the `playerlist.garou3299.json` file.

**Option B:** Go [here](https://github.com/Garou3299/TF2BD-Database/blob/master/playerlist.garou3299.json), click the `More file actions` button in the top-right corner, and download the `playerlist.garou3299.json` file directly.

**Q:** *How I can contribute to your Database?*

**A:** If you have a cheater’s permanent Steam profile link or ID along with video evidence proving the behavior, please submit it [here](https://github.com/Garou3299/TF2BD-Database/discussions/1).
